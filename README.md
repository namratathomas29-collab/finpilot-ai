# FinPilot AI

FinPilot AI is an AI-powered financial planning and expense tracking platform built using Spring Boot, React, JWT Authentication, and Recharts.

## Features

- User Registration & Login
- JWT Authentication
- Expense Tracking
- Financial Goal Management
- AI-Powered Financial Insights
- Spending Analytics
- Interactive Charts & Visualizations
- Personalized Recommendations

## Tech Stack

### Frontend
- React.js
- JavaScript
- HTML/CSS
- Recharts

### Backend
- Spring Boot
- Java
- Spring Security
- JWT Authentication
- JPA/Hibernate

### Database
- H2 Database
